Introduction

Minora is an elegant minimalist UI kit built with vanilla HTML, CSS, and JavaScript. Every component uses CSS custom properties from a single source of truth — change one token, the entire system adapts.

Installation

Add Minora to your project via CDN. No build step required.

<!-- CSS: all components -->
<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/gh/ilzamafif/minora@v0.1.1/dist/minora.min.css">

<!-- JS: modal, toast, select, tooltip -->
<script src="https://cdn.jsdelivr.net/gh/ilzamafif/minora@v0.1.1/dist/minora.min.js"></script>

Quick Start

Use any component by adding the appropriate classes to your HTML.

Button

Interactive element for actions, navigation, and form submission. Available in solid, outline, and ghost variants with three sizes and loading state.

Variants

Solid

btn btn-primary|secondary|neutral btn-md 
<button class="btn btn-primary btn-md">Primary</button>
<button class="btn btn-secondary btn-md">Secondary</button>
<button class="btn btn-neutral btn-md">Neutral</button>

Outline

btn btn-outline-primary|secondary|neutral btn-md 
<button class="btn btn-outline-primary btn-md">Outline</button>
<button class="btn btn-outline-secondary btn-md">Outline</button>
<button class="btn btn-outline-neutral btn-md">Outline</button>

Ghost

btn btn-ghost-primary|secondary|neutral btn-md

Destructive

Sizes

btn-sm (32px) · btn-md (40px) · btn-lg (48px)

States

With Icon

<button class="btn btn-primary btn-md">
  <svg width="16" height="16" viewBox="0 0 24 24">...</svg>
  Download
</button>

API

Class Type Default Description
btn string required Base button reset and layout
btn-{variant} string — Color scheme: primary, secondary, neutral, outline-*, ghost-*, destructive*
btn-{size} string btn-md Height: btn-sm (32px), btn-md (40px), btn-lg (48px)
btn-icon boolean false Make button circular for icon-only use
btn-loading boolean false Show spinner, disable interactions

Toast

// Simple toast

  type: 'success',
  message: 'Changes saved!'
});

// With action button

  type: 'error',
  message: 'Upload failed.',
  action: {
    label: 'Retry',
    onClick: () => /* retry logic */
  }
});
