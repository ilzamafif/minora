Why Minora?

Built for developers who value elegance and simplicity

Zero Dependencies

Pure vanilla HTML, CSS, and JavaScript. No React, no Tailwind build step, no npm packages to install. Just link the CSS and go.

Token-Driven Design

Every color, spacing, and animation comes from CSS custom properties. Change one token and the entire system adapts instantly.

CDN Ready

Available via jsDelivr CDN. One link tag in your HTML and you have access to all 18 components, fully built and tested.

Responsive by Default

Every component is built to work beautifully on any screen size. No extra responsive utilities needed — just use the classes.

Accessible

ARIA attributes, keyboard navigation, focus-visible rings, and screen reader support built into every interactive component.

MIT Licensed

Free for personal and commercial use. No attribution required, though always appreciated. Build something great.
