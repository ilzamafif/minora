Why Minora?

Built for developers who value elegance and simplicity

Zero Dependencies

Pure vanilla HTML, CSS, and JavaScript. No React, no Tailwind build step. Just link the CSS and go.

Token-Driven Design

Every color, spacing, and animation from CSS custom properties. Change one token — the entire system adapts.

CDN Ready

Available via jsDelivr. One link tag and you have all 18 components, fully built and tested.

Responsive by Default

Every component works beautifully on any screen size. No extra responsive utilities needed.

Accessible

ARIA attributes, keyboard navigation, focus-visible rings — built into every interactive component.

MIT Licensed

Free for personal and commercial use. No attribution required, though always appreciated.
